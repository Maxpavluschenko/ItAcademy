About me

I'm a web developer with 1 years of professional experience. In my works I always try to combine aesthetic with functional aspects, looking for the right balance to communicate without annoying.

What makes a great website? Here's my opinion: a successful website should be attractive but easy to read. A fascinating message, together with an immediate understanding. Original content and easily traceable information.

My Skills

For frontend developer one should know following topics:

  • HTML :Must know what are html tags and how to handcraft HTML templates.
  • CSS :While hand crafting template you should require css to make more attractive UI design.
  • DOM :Document object model and how document is created. If you have good jquery background you can easily pick up DOM manipulations.
  • Object Oriented JavaScript: Global namespace: All JS Framework heavily uses javascript name space.
  • Object Oriented JavaScript: Inheritance:Inheritance is very important concept in JavaScript. Inheritance is heavily used in all the frameworks in JavaScript.
  • Closures and Prototype:Closures and Prototype is heavily used in all JavaScript frameworks.
